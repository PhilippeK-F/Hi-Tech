HiTech.NET — Site E-commerce High-Tech

Description du projet :

Site e-commerce spécialisé dans le matériel high-tech (ordinateurs, périphériques, téléphones, objets connectés), développé en HTML5, CSS3 et JavaScript orienté objet. Le projet comprend une fiche produit complète côté client et un back-office d'administration Bootstrap.

Fonctionnalités :

- Header avec barre de recherche, menu compte et panier
- Navigation multi-niveaux (mega menu) : Ordinateurs, Périphériques, Téléphones, Objets connectés
- Options d'accessibilité : thème sombre / clair, interlignage augmenté
- Galerie photos interactive avec changement d'image au clic
- Schéma technique Canvas : représentation graphique de l'écran avec taille et ratio
- Fiche technique complète du produit

Panier interactif (localStorage) :

- Ajout / suppression de produits
- Compteur et total mis à jour en temps réel
- Persistance des données entre les sessions


Liste de souhaits :

- Modal pour la garantie ou la fiche détaillée
- Balises Open Graph (partage réseaux sociaux)
- SEO : meta description, title, favicon

Back-office admin (Bootstrap) :

- Sidebar de navigation (Dashboard, Rubriques, Produits, Commandes, Paramètres)
- Liste des produits avec tableau responsive
- Formulaire d'ajout / modification de produit
- Interface responsive Bootstrap 4

JavaScript Orienté Objet — Classes :

- Cart (cart.js)
- Gestion complète du panier avec persistance localStorage.
- Méthode Description
- Galerie photos interactive avec changement d'image au clic.
- Modal (modal.js)
- Fenêtre modale avec ouverture/fermeture et focus automatique.
- Canvas (canvas.js)
- Rendu graphique d'un schéma d'écran (taille en pouces, ratio, diagonale) via l'API Canvas HTML5.

Stack technique :
Outil Usage HTML5 Structure sémantique, accessibilité, Open GraphCSS3Mise en page, thèmes, responsive JavaScript ES6+Classes, localStorage, Canvas API, DOMBootstrap 4Back-office adminFont, front-end personnalisées.

Points techniques notables :

- JavaScript 100% orienté objet — 3 classes indépendantes et réutilisables
- Panier persistant — les articles restent en mémoire après fermeture du navigateur
- Canvas API — schéma technique généré dynamiquement en JavaScript pur
- Accessibilité — navigation clavier, thème contrasté, interlignage ajustable
- SEO & Open Graph — balises optimisées pour le partage et le référencement
